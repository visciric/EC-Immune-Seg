## Exemplary Results

To illustrate the output of the pipeline, a set of example heatmaps has been added to the repository. These visualizations show the spatial distribution of immune cell density across tumor tiles, as predicted by HoVer-NeXt on representative Whole Slide Images — organized by tumor subtype.

The heatmaps color-encode local immune cell density, highlighting regions of high immune infiltration versus immune-cold areas within the tumor microenvironment.

**Example outputs are organized by subtype:**

```
results/
├── msi-h/        # Exemplary heatmaps for MSI-H cases
├── mss_tmb-l/    # Exemplary heatmaps for MSS/TMB-L cases
└── tmb-h/        # Exemplary heatmaps for MSS/TMB-H cases
```

Each folder contains 5 representative heatmaps intended as a visual reference for expected pipeline output.
