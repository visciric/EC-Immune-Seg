# Data Directory Documentation



## Directory Structure

```
data/
├── processed_csv/              # Generated files from analysis notebooks
│   ├── combined_wsi_data.csv   # Master dataset: all tiles with features
│   ├── case_level_aggregated.csv  # Case-level summary statistics
│   ├── msi-h.csv               # MSI-H subgroup tiles
│   ├── mss_tmb-h.csv           # MSS/TMB-H subgroup tiles
│   ├── mss_tmb-l.csv           # MSS/TMB-L subgroup tiles
│   └── *.zip                   # Compressed versions of CSV files
├── raw/                        # Image lists for model inference
│   ├── msi-h_image_list.csv
│   ├── mss_tmb-l_image_list.csv
│   └── tmb_h_image_list.csv
├── stad_clinical.csv           # STAD (stomach) clinical data
├── ucec_clinical.csv           # UCEC (endometrial) clinical data (primary)
└── README.md
```



## Data Flow

```
┌─────────────────────────────────────────────────────────────────────┐
│                    DATA PROCESSING PIPELINE                         │
└─────────────────────────────────────────────────────────────────────┘

1. CLINICAL DATA (Starting Point)
   ├── ucec_clinical.csv ──→ Contains case IDs and WSI image connections
   └── stad_clinical.csv     (includes MSI_TMB, patient info, survival)
                │
                ↓
2. RAW IMAGE LISTS
   └── raw/*.csv ──→ Links to actual WSI images stored elsewhere
                     (used for model inference to know which files to process)
                │
                ↓
3. MODEL INFERENCE
   └── Segmentation model processes WSI images
       (generates tile-level predictions)
                │
                ↓
4. PROCESSED DATA 
   └── processed_csv/
       ├── combined_wsi_data.csv ─→ All tiles with segmentation features
       ├── msi-h.csv, mss_tmb-h.csv, mss_tmb-l.csv ─→ Subgroup-specific tiles
       └── case_level_aggregated.csv ─→ Aggregated case-level statistics
```

### Key Points:
- **ucec_clinical.csv** is the **starting point**, it contains patient information and the link between case IDs and whole slide images (WSI)
- **raw/ folder** contains image lists used to tell the segmentation model which WSI files to process (images are stored in a separate location)
- **processed_csv/ folder** contains files generated by analysis notebooks after model inference



## Clinical Data

### Files:
- `ucec_clinical.csv` - Uterine Corpus Endometrial Carcinoma
- `stad_clinical.csv` - Stomach Adenocarcinoma

These files contain patient-level clinical information linked to WSI images.


## Raw Data

**Location:** `raw/`

Contains CSV files with lists of WSI image paths used for model inference:

- `msi-h_image_list.csv` - Paths to MSI-H cases
- `tmb_h_image_list.csv` - Paths to MSS/TMB-H cases
- `mss_tmb-l_image_list.csv` - Paths to MSS/TMB-L cases

### Purpose:
These files act as pointers to the actual WSI image files stored in a separate location (not in this repository). The segmentation model reads these lists to know which images to process during inference.

### Structure:
Each file typically contains:
- Image file paths
- Case identifiers


## Processed Data

**Location:** `processed_csv/`

Contains analysis results generated by notebooks after model inference on WSI images.

### Main Files:

#### combined_wsi_data.csv
**Master dataset containing all tile-level information:**
- Tile coordinates and metadata (tile_id, x_start, y_start, width, height)
- Segmentation results (neoplastic, inflammatory, connective, necrosis, non_neoplastic cell counts)
- Percentage and normalized features (e.g., inflammatory_pct, neoplastic_normalized)
- Case identifiers linking to clinical data
- Molecular subgroup (MSI_TMB column: MSI-H, MSS/TMB-H, MSS/TMB-L)

#### case_level_aggregated.csv
**Aggregated statistics at the patient case level:**
- One row per patient case
- Summary statistics (mean, median, std) of tile features
- Clinical information merged from ucec_clinical.csv
- Used for case-level analysis and visualization

#### Subgroup-specific Files:
- `msi-h.csv` - Tiles from MSI-H cases only
- `mss_tmb-h.csv` - Tiles from MSS/TMB-H cases only
- `mss_tmb-l.csv` - Tiles from MSS/TMB-L cases only

These are filtered subsets of combined_wsi_data.csv for subgroup-specific analysis.
